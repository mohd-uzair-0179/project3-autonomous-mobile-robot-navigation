## Overview

This project simulates an autonomous mobile robot navigating through a 2D environment.

The robot performs:

- Occupancy Grid Mapping
- Simulated LiDAR Scanning
- A* Path Planning
- Dynamic Obstacle Avoidance

---

## Features

- Occupancy Grid Map
- Simulated LiDAR
- A* Algorithm
- Obstacle Detection
- Robot Navigation
- Visualization
